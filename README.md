# pangcg-components

A react library developed with dumi

## 安装

使用你喜欢的包管理器安装：

:::code-group

```bash [npm]
npm install pangcg-components --save
```

```bash [cnpm]
cnpm install pangcg-components --save
```

```bash [yarn]
yarn add pangcg-components --save
```

```bash [pnpm]
pnpm add pangcg-components --save
```

:::
